github建立博客

Posted on 2018-09-03 | In 开发笔记 , 其他
github结合Hexo建立静态博客

准备工作

  1. 安装配置node、npm
  2. 注册github账号下载github客户端、git工具 https://desktop.github.com/
  3. github上创建博客仓库
  4. 打开github客户端下载代码

构建站点项目

安装

  1. 创建站点文件夹，进入站点文件夹，右键”git bash”
  2. 安装hexo “npm install hexo-cli -g”
  3. 初始化 “hexo init”
  4. 安装依赖 “npm install”

配置

打开站点配置文件，参考官网配置 https://hexo.io/zh-cn/docs/configuration

部署

  • hexo g -编译
  • hexo s -预览
  • hexo c -清除缓存

主题

示例：nexT主题

  • 下载主题包 https://github.com/iissnan/hexo-theme-next/releases
  • 解压压缩包重命名为next,拷贝至theme文件夹
  • 站点配置文件配置theme: next
  • 主题配置文件 _config.yml. http://theme-next.iissnan.com/getting-started.html

写作

新建文章

hexo new [layout] <title>

layout

  • post 默认布局
  • page
  • draft 草稿
    新建文章布局会根据scaffolds文件夹下对应文件的内容建立文章

Front-Matter

草稿

将草稿移至_posts
hexo publish [layout] <title>
